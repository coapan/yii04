1、Advanced 模板的安装
-------------------
**Composer安装**

_Windows 下，先更新，执行: `composer self-update`_

_下载项目文件，执行: `composer create_project --prefer-dist yiisoft/yii2-app-advanced yii04`_

_初始化：`php init`_

_数据库迁移：`php yii migrate`，这样就自动生成了一张 user 表_
